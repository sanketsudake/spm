Snooker Player Profile Management
----------------------------

How-To
--------------------
Compile code
```
$ git clone https://github.com/tripples/spm.git ; cd spm
$ git checkout your_tag
$ mkdir build ; cd build
$ cmake ..
$ make
$./snooker video_Arg
```
